# VERSPA Brand Assets

## Logo mark — the V
A single continuous path forming the letter V. Two descending strokes meet at the midpoint and rise. The teal dot below anchors the mark. Reads as: signal, transmission, energy pulse.

## Files
- `favicon-32.svg` — 32×32 favicon (no dot at this scale)
- `favicon-64.svg` — 64×64 retina favicon
- `logomark-dark-200.svg` — 200×200, dark background (primary use)
- `logomark-light-200.svg` — 200×200, white background (reversed)
- `logomark-teal-200.svg` — 200×200, teal background (accent variant)
- `social-1200.svg` — 1200×1200 social profile image
- `social-banner-1500x500.svg` — 1500×500 LinkedIn/Twitter banner
- `wordmark-horizontal.svg` — Full wordmark with icon + VERSPA + tagline

## Colours
- Ink: `#0B0C0E`
- Teal: `#0A6648`
- White: `#FFFFFF`

## Usage
All files are SVG — scale to any size without quality loss. For web use, embed the SVG inline or reference as `<img>`. For favicon, convert `favicon-32.svg` to `.ico` using a tool like RealFaviconGenerator.
