# VERSPA Landing Page v2

Production-ready landing page redesign. White/black editorial aesthetic inspired by Zynta.com. Deploy `index.html` directly — single file, no build step.

## Design Direction
- Predominantly white with black editorial typography
- Cormorant Garamond (serif display) + Outfit (sans body)
- Teal as single surgical accent colour
- Numbered feature callouts (01, 02, 03)
- Authority-driven stat callouts
- Scrolling marquee ticker strip
- No particle animations — clean and professional

## Sections
1. Hero — stats card + dual CTAs
2. Marquee strip — rotating platform keywords
3. Problem — two-panel numbered problem statement
4. Platform — numbered feature list (all 6 pillars)
5. Who We Serve — 6-card audience grid (dark background)
6. Diaspora Capital Rails — stats visual + feature list
7. Why London — 6-card grid
8. Waitlist — dual-tab form (Investor / Project Owner)
9. Footer — full navigation + FCA disclaimer

## Deploy
Drop `index.html` onto Netlify Drop, Vercel, or any static host.
